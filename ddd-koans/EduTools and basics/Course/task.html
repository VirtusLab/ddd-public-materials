Course Structure

All courses available in EduTools has the same tree-like structure:

  • Course
    • Lesson 1
        Task 1
        Task 2
        ...
    • Lesson 2
        Task 1
        Task 2
        ...
    • ...

Additionally, lessons can be grouped in sections. However, this course doesn't use sections.

The course structure can be viewed/browsed in the left bar in the Intellij window
