EduTools plugin

Plugin

Jetbrain's Educational Products are set of tools that can be used for teaching and learning programming in form of various different tasks.

The EduTools plugin is required to start course.
Thanks to that plugin, many courses can be browsed and run in the Intellij IDE itself.

Working with the course

There are three main parts of the Intellij's window that you will interact with:

  • Course structure on the left
  • Code editor in the middle
  • Task description on the right

Additionally, on the bottom of the right sidebar with the task description, there are two button Run and Next as you can in the below picture.
Please click on the Next button to go to the next task or Run to execute code available in the code editor.

Each will be described separately in next tasks in the EduTools and basics lesson.

Don't worry about lessons and task now, if you want to learn about them just press Next button. You can also write any code in the middle and then press run to make it work.
