To know what is represented with the values we need to check properties' names. So we need a rescue here. The concept that will help us is called called tiny types.

So tiny type is very small and simple value object with just one property. Typically it wraps primitive type. In this task you will have to find right names for two tiny types, add them as private property in ValidTimeSignature (maintaining immutability) and write create factory method again.
