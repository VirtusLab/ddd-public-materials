Immutability means that object's state cannot be changed after creation.

In this task and at least few next you will develop TimeSignature class as it could serve as a great example of value object. So here we go with reminder of what they are.

Time signatures are symbols which describe repeating beat rhythm in a piece of music. It tells us the total length of all symbols in a bar. In text time signatures are written using the slash sign "/" like 3/4. The upper value is called numerator and lower denominator.

As the description states TimeSignature must contain only two properties:
  • nominator,
  • denominator,
both of them should be of type Int and declared using val not var.

Doing it that way will ensure immutability of thr TimeSignature.

Remember not to use nullable type Int?.
