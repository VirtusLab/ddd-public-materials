TimeSignature is described using two properties, the same you have declared in previous task.
However, values of this properties cannot be random integers. There are domain rules. If you don't remember, here you go:

  • numerator is a number between 1 and 32,
  • denominator is a number between 1 and 32,
  • denominator is a number of power of two.

How should we validate the values? Let your code throw an exception of any kind on invalid values
Consider using require function.

To check if number is power of two we provided extension functions Int.main.isPowerOfTwo().

This may not be obvious at first glance, but if you correctly check if denominator is power of two, then checking if denominator is between 1 and 63 would make test pass as values between 32 and 63 are covered by the power of two condition.
