Let's write a time signature of 1/2 at the beginning - says conductor.

In text time signatures are written using the slash sign "/" like "3/4".
It is an understandable notation for Musicians which can be easily followed in our model.

Do you think that musicians would understand such snippet TimeSignature.of("1/2")?

Hopefully, they do!
Let's then implement such feature inside TimeSignature value object.

You have to create of method that creates TimeSignature from String such as "1/2" or "4/4". The method call should look like this TimeSignature.of("1/2") and in the given case create ValidTimeSignature(1,2). Please create the given method in such manner that it accepts just the minimal acceptable string representation of time signature, any additional whitespaces (like " 1/2", "4/4     ") should return InvalidTimeSignature.
