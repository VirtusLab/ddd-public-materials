How to measure the real length of note value

As was stated in the introduction, the note value represents length in abstract unit of beats. Therefore, to get to know the real length in time, the beats must be translated to time unit and the beat must be connected to corresponding note value.

In case of the beat matching the note value, the default assumption is 1 beat is equal to length of quarter note value and this will be only mapping in the course. So remember 1 beat = length of quarter note value. However it is possible to use eight note value or half note value as equal to one beat.

In music the concept of this translation is called tempo. It represents number of beats in the minute. The class was already implemented for you, please take a look. The 'bpm' is well known and widely used shortcut for beats per minutes.

Example how to calculate length:

  • Note value: eighth note value
  • Tempo: 30 bpm
  • Target time unit: seconds
Steps:

  1. As 1 beat equals value of quarter note value and eighth note value lasts half of quarter note value, then eighth note value has value equal to 0.5 as 1 beat(s) / 2 = 0.5 beats.
  2. As tempo is 30 beats/minute, then to obtain the value in minutes, we need to divide, the eight note value in beats by the tempo value.
    0.5 beats / (30 beats/minute) = (1/60) minute
  3. To obtain the time value in given unit, we must change the unit from minutes to seconds. As we all know, one minute is 60 seconds. Therefore: (1/60) minute * 60 (seconds/minute) = 1 s

Now we can provide the answer, the length on eighth note value in tempo 30 bpm is 1 second.
