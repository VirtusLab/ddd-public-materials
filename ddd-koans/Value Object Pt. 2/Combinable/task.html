Combinable property

The operator fun plus(other: Length): Length was added, this is not that trivial as it seems. In the case of length, it is very natural to add one to another, however, it is worth considering how you can combine other value objects. This doesn't have to be adding, it can be any way to bind or bond values together.

The possibility to combine value objects is called Combinable property. It is connected with behavior-rich property, as being combinable extends the behavior of the value object.
